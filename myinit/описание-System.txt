02.

sudo hostname new_name

1вариант:
/etc/hostname

2вариант:

04.

05.
Назовите команду, к-рая определяет состояние службы SSH.

1вариант:

2вариант:
systemctl status ssh.service

3вариант:

4вариант:

06.

1вариант:

2вариант:
sudo systemctl restart ssh.service

3вариант:
 sudo systemctl restart sshd.service


07.
Выясните PID службы SSHD.

1вариант:
cat /var/run/sshd.pid

2вариант:
pidof sshd | cut -d' ' -f1

3 вариант:
pgrep -u root sshd

4вариант:
sudo service ssh status | grep PID | cut -d ' ' -f 4

11.
Назовите команду, к-рая отображает доступное оставшееся пространство и используется в системе способом, понятным для человека.

1вариант:
df -h

2вариант:
df -h --total

3вариант:
df -Hl --total

12.
Выясните точный размер каждой папки /var в понятной для человека форме, следуя по пути к ней.

1вариант:
du -h --max-depth=1 /var

2вариант:
du -h /var/*

3вариант:
sudo du -h -s /var/*

4вариант:
sudo du -h -d 1 /var

19.
Перечислите все службы, к-рые запускаются во время загрузки, и назовите этот вид служб.

1вариант:
systemctl status --type service --all

2вариант:
systemctl list-unit-files --type=service | grep "enabled" | awk -F'.service' '{print $1}'

20.
Перечислите всех существующих пользователей на виртуальной машине.

cat /etc/passwd | cut -d ":" -f 1

21.
Перечислите всех реальных пользователей на виртуальной машине.

cat /etc/passwd | egrep ':[0-9]{4}' | cut -d: -f1
